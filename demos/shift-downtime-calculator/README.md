# Shift Downtime Calculator

A small portfolio demo for operations workers who need clean end-of-shift downtime reports.

## What It Does

- Adds break, lunch, EBB, charging, troubleshooting, and other downtime.
- Converts downtime minutes into decimal hours.
- Adjusts totals when charging overlaps with break or lunch.
- Generates a copy-ready EOS report.
- Saves the last entry in the browser with `localStorage`.

## How To Open It

Open `index.html` in a browser.
