# Tip Split Calculator | Philip Builds Studio

A simple browser tool for splitting a restaurant bill and tip.

## Features
- Bill total input
- Tip percentage quick-pick buttons (10/15/18/20/25%)
- Number of people
- Live calculated: tip amount, total, per-person cost

## Notes
No data saved or sent. Pure HTML/CSS/JS, no dependencies.
