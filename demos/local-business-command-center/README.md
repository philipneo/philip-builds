# Local Business Command Center | Philip Builds Studio

Flagship interactive demo — a fictional daily command center for local service businesses.

## Sections
- **Hero** — dark premium hero with animated dashboard preview and floating stat cards
- **Dashboard story** — sticky scroll section showing the before/during/after workflow
- **Quote estimator** — live calculator (service type, size, add-ons, urgency)
- **Daily checklist** — 7-item workflow tracker with animated progress bar
- **End-of-day summary** — auto-generated recap card with copy-to-clipboard button
- **What this proves** — honest proof section, no fake claims

## Technical
- Pure HTML/CSS/JS, no dependencies, no external requests
- IntersectionObserver scroll-reveal system
- CSS keyframe animations (float, glow pulse)
- Sticky story section (desktop only, collapses on mobile)
- Live quote calculation with NaN-safe arithmetic
- Clipboard API with execCommand fallback
- Full reduced-motion support

## Notes
All content is fictional. No real clients, revenue figures, or business results.
