# Cleaning Quote Calculator | Philip Builds Studio

Instant home cleaning price estimator for service businesses.

## Features
- Home size selector (sq footage brackets)
- Bathroom count
- Service type (standard / deep clean / move-in)
- Add-on checkboxes (oven, fridge, windows, laundry, cabinets)
- Live price range with itemized breakdown

## Notes
No data saved or sent. Pure HTML/CSS/JS, no dependencies.
