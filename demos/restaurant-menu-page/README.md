# Restaurant Menu Page | Philip Builds Studio

A styled, mobile-friendly restaurant menu page demo.

## Features
- Hero section with restaurant branding
- Sticky section nav with scroll-spy active state
- 5 menu sections: Starters, Mains, Sides, Desserts, Drinks
- Dotted price leaders, item descriptions, house-favorite badges
- Fictional brand disclaimer

## Notes
Casa Moreno Kitchen is a fictional demo restaurant. Pure HTML/CSS/JS, no dependencies.
