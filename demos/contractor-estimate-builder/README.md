# Contractor Estimate Builder | Philip Builds Studio

A browser-only estimate builder for contractors and trade businesses.

## Features
- Business name, license/contact, client name, project description
- Add/remove line items with quantity and unit price
- Markup/overhead percentage
- Live preview with itemized totals
- Print / Save PDF and copy summary

## Notes
No data saved or sent. Pure HTML/CSS/JS, no dependencies.
