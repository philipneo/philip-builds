# Simple Booking Request | Philip Builds Studio

A demo booking request form for local service businesses.

## Features
- Name, phone, service type, preferred date/time, address, notes
- Demo success state on submit (no data sent)
- Trust signals on the left panel
- Reset and re-submit flow

## Notes
No data saved or sent. Pure HTML/CSS/JS, no dependencies.
