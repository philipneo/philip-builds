# Invoice Builder | Philip Builds Studio

A browser-only invoice builder for freelancers and small service businesses.

## Features
- Business name, client name, invoice number, date
- Add/remove line items with live totals
- Optional tax/fee percentage
- Live invoice preview panel
- Print / Save PDF button
- Copy summary to clipboard

## Notes
No data saved or sent. Pure HTML/CSS/JS, no dependencies.
