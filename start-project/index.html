How projects are scoped

Three rough shapes. Real scope depends on the job.

These are starting points, not fixed quotes — tell me what you need and I'll give you an honest fit.

Starter site

A simple presence

  • One-page landing layout
  • Mobile-first structure
  • Clear contact path + basic copy

Best for a clean first site.

Interactive demo / tool

A working front-end tool

  • Custom UI + front-end logic
  • Responsive, satisfying interactions
  • No backend unless future scope

Best for calculators, quotes, invoices, booking-style flows.

Studio system

A deeper multi-page build

  • Homepage + case-study sections
  • Reusable components + guide
  • Full polish pass

Best for a site that should feel like one product.
